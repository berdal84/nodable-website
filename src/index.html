Nodable is node-able

The goal of Nodable is to provide an original hybrid source code editor, using both textual and nodal paradigms.

By constrast with other nodal softwares, Nodable allows the user to edit the text and the graph at any time...

How does it works?

Nodable is able to generate a graph from a source code and a code from a graph.

Any changes to the code will update the graph.

If we replace +10 by -7 from the code above, Nodable will update the graph accordingly.

2) Any changes to the graph will update the code.

If we change the 7 by 2.5, we connect it to a divide operator with the 5 and we remove the multiply operator, Nodable will update the code accordingly.

There is no magic, the source code stays the source code. The graph is always generated on the fly.
